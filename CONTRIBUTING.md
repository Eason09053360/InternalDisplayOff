# Contributing

Thanks for taking a look at Internal Display Off.

Please keep the app small and focused. The goal is one reliable action: turn off the built-in MacBook display when an external display is connected, then make restoration obvious.

Before opening a pull request:

1. Run `./build.sh`.
2. Test with an external display connected.
3. Confirm the menu bar restore action turns the internal display back on.
4. Note your macOS version and Mac model in the pull request.

Because this project uses a private macOS display function, compatibility reports are especially useful.
